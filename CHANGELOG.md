# Changelog

Version numbers (`v0.<n>`) are the commit count, as shown in the app
header — see the Versioning section of the README.

## v0.10 — 2026-07-13

- Added this changelog.

## v0.9 — 2026-07-13

- Commit-based version tag in the app header (`v0.<n> · <date>`),
  regenerated by a pre-commit hook.

## v0.8 — 2026-07-13

- All inputs (course, wind, current, mode toggles) are persisted in
  localStorage, so a simulation set up once is still there on the
  next visit.

## v0.7 — 2026-07-13

- The playback bar moved to the bottom-right of the chart; it no
  longer covers the scale bar.

## v0.6 — 2026-07-13

- Bearing inputs (mark bearing, mean wind, absolute-TWD phases) are
  constrained to the 0–359 compass range and wrap on blur
  (450 → 90); phase offsets wrap to ±180.

## v0.5 — 2026-07-13

- Fixed one-screen layout: the controls and stage columns end on the
  same bottom line, each scrolling internally when taller than the
  viewport. The chart sizes itself to the space actually free in the
  stage column.

## v0.4 — 2026-07-13

- Split the single-file app into `index.html`, `css/style.css` and
  `js/app.js`.

## v0.3 — 2026-07-13

- MIT license.

## v0.2 — 2026-07-12

- Renamed the project to Sailing Upwind Tactics Simulator.

## v0.1 — 2026-07-12

- Initial release: upwind tactics simulator with current field
  interpolation, wind scenarios, boat polar library and playback.
